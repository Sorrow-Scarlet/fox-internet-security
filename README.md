# 使用

## 下载

- 对于普通用户

  1. 找到 Release
  2. 下载最新的，符合自己操作系统的版本
  3. 解压并安装

- 对于开发者

```
git clone https://github.com/Sorrow-Scarlet/fox-internet-security
```

- 对于源码编译用户

```
git clone --depth 1 https://github.com/Sorrow-Scarlet/fox-internet-security
```
